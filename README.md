# Hugo Garcés 50

Invitación digital inspirada en Fórmula 1 para celebrar los 50 años de Hugo Garcés.

## Publicar directamente en GitHub Pages

Puedes subir todo el contenido de esta carpeta a la raíz del repositorio.
`index.html` debe quedar al mismo nivel que las carpetas `css`, `js` y `assets`.

## Ejecutar localmente con Vite

Requiere Node.js 20.19 o superior.

```bash
npm install
npm run dev
```

## Generar producción

```bash
npm run build
```

## Cambios integrados

- Hero horizontal para escritorio y 9:16 para móvil.
- Todas las imágenes principales usan extensión `.png`.
- Fondos LEGO en Fecha, Hora y Lugar.
- Iconos de calendario, reloj y ubicación.
- Menú sin la tarjeta de Hidratación.
- Botón de confirmación por WhatsApp.
- Número de WhatsApp: +52 729 159 9530.
- Mensaje precargado: “Quiero confirmar mi asistencia al cumpleaños de Hugo”.

## Nota sobre el formulario

El formulario HTML muestra una confirmación visual, pero no almacena datos en un servidor.
El botón de WhatsApp sí abre una conversación real con el mensaje precargado.
