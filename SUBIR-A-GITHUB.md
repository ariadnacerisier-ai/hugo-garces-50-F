# Cómo subir esta versión a GitHub

1. Descomprime el ZIP.
2. Abre la carpeta de tu repositorio local.
3. Sustituye todo su contenido por los archivos de esta carpeta.
4. Confirma que `index.html` esté en la raíz.
5. Haz commit con el mensaje: `Actualización completa de la invitación`.
6. Haz push a la rama `main`.
7. Espera a que GitHub Pages termine el despliegue.
8. Recarga sin caché.

No subas la carpeta exterior como una subcarpeta adicional. Debe quedar así:

```text
hugo-garces-50/
├── index.html
├── css/
├── js/
├── assets/
├── package.json
└── README.md
```
